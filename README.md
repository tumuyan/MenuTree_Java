这是一个用java写的小程序，主要用于生成文件树，比如：

        com
         +- example
             +- myapplication
                 +- Application.java
                 |
                 +- customer
                 |   +- Customer.java
                 |   +- CustomerController.java
                 |   +- CustomerService.java
                 |   +- CustomerRepository.java
                 |
                 +- order
                     +- Order.java
                     +- OrderController.java
                     +- OrderService.java
                     +- OrderRepository.java
